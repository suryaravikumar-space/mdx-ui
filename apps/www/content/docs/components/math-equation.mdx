```bash
npx @ravikumarsurya/mdx-ui add math-equation
```

## Equation

A centered display block with an optional equation number. Compose with `M`/`BM` for LaTeX expressions, or with [Math Primitives](/docs/components/math-primitives) for structured layout.

```mdx
<Equation label="1">
  <BM expr="E = mc^2" />
</Equation>
```

<Equation label="1">
  <BM expr="E = mc^2" />
</Equation>

### With primitives

```mdx
<Equation label="2">
  x = <Frac
    num={<>-b <PlusMinus /> <Sqrt>b² - 4ac</Sqrt></>}
    den="2a"
  />
</Equation>
```

<Equation label="2">
  x = <Frac num={<>-b <PlusMinus /> <Sqrt>b² - 4ac</Sqrt></>} den="2a" />
</Equation>

### No label

Omit `label` for an unlabeled display block:

```mdx
<Equation>
  <M expr="\nabla^2 \phi = 0" />
</Equation>
```

<Equation>
  <M expr="\nabla^2 \phi = 0" />
</Equation>

### Inline mode

Use `inline` to wrap expressions inside prose without the outer box:

```mdx
The identity <Equation inline>e<Pow exp={<>iπ</>} /> + 1 = 0</Equation> connects five constants.
```

The identity <Equation inline>e<Pow exp={<>iπ</>} /> + 1 = 0</Equation> connects five fundamental constants.

## EqSystem

A system of simultaneous equations with an SVG left brace.

```mdx
<EqSystem>
  <div>x + y = 5</div>
  <div>2x − y = 1</div>
</EqSystem>
```

<EqSystem>
  <div>x + y = 5</div>
  <div>2x − y = 1</div>
</EqSystem>

### With LaTeX expressions

```mdx
<EqSystem>
  <div><M expr="\frac{\partial u}{\partial t} = \alpha \nabla^2 u" /></div>
  <div><M expr="u(x, 0) = f(x)" /></div>
  <div><M expr="u(0, t) = u(L, t) = 0" /></div>
</EqSystem>
```

## Props

### Equation

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | Equation number/label shown right-aligned as `(1)` |
| `inline` | `boolean` | `false` | Inline span with no outer box |
| `children` | `ReactNode` | — | Math content |
| `className` | `string` | — | Additional CSS classes |

### EqSystem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | One element per equation |
| `className` | `string` | — | Additional CSS classes |
