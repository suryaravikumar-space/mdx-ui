## When to use

Primitives are best for **structured math UI** ‚Äî numbered equations with layout, solution blocks, inline symbols alongside prose. For writing raw expressions, `<M expr="..." />` (LaTeX string) is simpler. Both can be mixed.

```mdx
{/* LaTeX string ‚Äî best for expressions */}
<M expr="\frac{-b \pm \sqrt{b^2-4ac}}{2a}" />

{/* Primitives ‚Äî best for layout and structure */}
<Equation label="1">
  x = <Frac num={<>-b <PlusMinus/> <Sqrt>b¬≤-4ac</Sqrt></>} den="2a" />
</Equation>
```

## Greek letters

`<Greek letter="alpha" />` for any letter. Shortcuts for all 23 lowercase + 10 uppercase:

```mdx
<Alpha />  <Beta />  <Gamma />  <Theta />
<Lambda />  <Mu />  <Pi />  {/* etc */}

{/* Uppercase */}
<GammaU />  <DeltaU />  <SigmaU />  <OmegaU />
```

Note: `GDelta` (Œ¥), `PiSym` (œÄ), `SigmaSym` (œÉ) avoid conflicts with other exports.
