## The single math system

Math primitives are the **only** math API in mdx-ui. Every component is pure React/CSS — no KaTeX, no LaTeX strings, no engine required.

Each component reads exactly like what it renders:

```mdx
<Frac num="a" den="b" /> → a/b fraction
<Pow exp="2">x</Pow> → x²
<Integral from="0" to="1">
  f(x) dx
</Integral>
→ ∫₀¹ f(x) dx
<Alpha /> → α<Angle>ABC</Angle> → ∠ABC
```

## Basic structural

| Component             | Renders           | Props            |
| --------------------- | ----------------- | ---------------- |
| `<Frac num den />`    | Vertical fraction | `num`, `den`     |
| `<Pow exp>base</Pow>` | base^exp          | `exp`, children  |
| `<Sub sub>base</Sub>` | base_sub          | `sub`, children  |
| `<Sqrt>x</Sqrt>`      | √x with overbar   | `n` for nth root |
| `<Abs>x</Abs>`        | \|x\|             | children         |
| `<Paren>x</Paren>`    | (x)               | children         |
| `<Deg>90</Deg>`       | 90°               | children         |
| `<Inf />`             | ∞                 | —                |

```mdx
<Frac num="1" den="2" />

<Frac
  num={
    <span>
      <Pow exp="2">x</Pow> + 1
    </span>
  }
  den="2a"
/>

<Sqrt n="3">27</Sqrt>

<Abs>x − y</Abs>
```

## Calculus

| Component                  | Renders            | Props                  |
| -------------------------- | ------------------ | ---------------------- |
| `<Integral>`               | ∫                  | `from`, `to`, children |
| `<ContourIntegral>`        | ∮                  | `from`, `to`, children |
| `<DoubleInt />`            | ∬                  | —                      |
| `<TripleInt />`            | ∭                  | —                      |
| `<Sum>`                    | Σ                  | `from`, `to`, children |
| `<Prod>`                   | Π                  | `from`, `to`, children |
| `<Lim sub="x → 0">`        | lim                | `sub`, children        |
| `<Deriv of="y" by="x" />`  | dy/dx              | `of`, `by`             |
| `<PDeriv of="f" by="x" />` | ∂f/∂x              | `of`, `by`             |
| `<Nabla />`                | ∇                  | —                      |
| `<Laplacian />`            | ∇²                 | —                      |
| `<Differential />`         | upright d          | —                      |
| `<Overbrace label="n">`    | ⏞ with label above | `label`, children      |
| `<Underbrace label="n">`   | ⏟ with label below | `label`, children      |

```mdx
<Integral from="a" to="b">
  f(x) <Differential />x
</Integral>

<Sum from="i=1" to="n">
  <Pow exp="2">i</Pow>
</Sum>

<Lim sub="h → 0">
  <Frac num={<span>f(x+h) − f(x)</span>} den="h" />
</Lim>
```

## Composing complex expressions

Wrap sub-expressions in `{<span>...</span>}` when passing them as JSX props:

```mdx
{/* Quadratic formula */}
x = <Frac
num={<span>−b <PlusMinus /> <Sqrt><Squared>b</Squared> − 4ac</Sqrt></span>}
den="2a"
/>

{/* Gaussian integral */}

<Integral
  from={
    <span>
      −<Inf />
    </span>
  }
  to={<Inf />}
>
  <Pow
    exp={
      <span>
        −<Squared>x</Squared>
      </span>
    }
  >
    e
  </Pow>{" "}
  <Differential />x
</Integral>
```
