## Installation

```bash
npx @ravikumarsurya/mdx-ui add math-primitives
```

## When to use

Primitives are best for **structured math UI** — numbered equations with layout, solution blocks, inline symbols alongside prose. For writing raw expressions, `<M expr="..." />` (LaTeX string) is simpler. Both can be mixed.

```mdx
{/* LaTeX string — best for expressions */}

<M expr="\frac{-b \pm \sqrt{b^2-4ac}}{2a}" />

{/* Primitives — best for layout and structure */}

<Equation label="1">
  x ={" "}
  <Frac
    num={
      <>
        -b <PlusMinus /> <Sqrt>b²-4ac</Sqrt>
      </>
    }
    den="2a"
  />
</Equation>
```

## Basic structural

| Component          | Renders          | Props                     |
| ------------------ | ---------------- | ------------------------- |
| `<Frac num den />` | num/den fraction | `num`, `den`              |
| `<Pow exp />`      | base^exp         | `exp`, `base` or children |
| `<Sub sub />`      | base_sub         | `sub`, `base` or children |
| `<Sqrt>`           | √x with overbar  | `n` for nth root          |
| `<Abs>`            | \|x\|            | children                  |
| `<Paren>`          | (x) scaled       | children                  |
| `<Deg>`            | 30°              | children                  |
| `<Inf />`          | ∞                | —                         |

```mdx
<Frac num="1" den="2" />

x<Pow exp="2" /> + y<Pow exp="2" />

<Sqrt n="3">27</Sqrt>

<Abs>x - y</Abs>
```

x<Pow exp="2" /> + y<Pow exp="2" /> = <Pow base="r" exp="2" />

<Frac
  num={
    <>
      1 + <Sqrt>5</Sqrt>
    </>
  }
  den="2"
/>

## Greek letters

`<Greek letter="alpha" />` for any letter. Shortcuts for all 23 lowercase + 10 uppercase:

```mdx
<Alpha /> <Beta /> <Gamma /> <Theta />
<Lambda /> <Mu /> <Pi /> {/* etc */}

{/* Uppercase */}

<GammaU /> <DeltaU /> <SigmaU /> <OmegaU />
```

Note: `GDelta` (δ), `PiSym` (π), `SigmaSym` (σ) avoid conflicts with other exports.
