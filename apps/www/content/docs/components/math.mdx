## Installation

```bash
npx @ravikumarsurya/mdx-ui add math
```

Add KaTeX CSS to your root layout:

```tsx
import "katex/dist/katex.min.css"
```

## Usage

```jsx
// Inline math
<InlineMath>E = mc^2</InlineMath>

// Block (display) math
<BlockMath>
  \int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
</BlockMath>
```

## Props

### Math / InlineMath / BlockMath

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| children | string | — | LaTeX expression |
| block | boolean | false | Display mode (Math only) |
| className | string | — | Additional CSS classes |
