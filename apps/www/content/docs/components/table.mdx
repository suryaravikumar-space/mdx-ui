## Installation

```bash
npx @ravikumarsurya/mdx-ui add table
```

## Usage

```jsx
<Table>
  <TableHeader>
    <TableRow>
      <TableHead>Name</TableHead>
      <TableHead>Type</TableHead>
      <TableHead>Default</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>variant</TableCell>
      <TableCell>string</TableCell>
      <TableCell>default</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

## Components

- `Table` — wrapper with overflow scroll
- `TableHeader` — `thead` with muted background
- `TableBody` — `tbody`
- `TableFooter` — `tfoot` with muted background
- `TableRow` — `tr` with hover state
- `TableHead` — `th` styled header cell
- `TableCell` — `td` body cell
- `TableCaption` — caption below the table
