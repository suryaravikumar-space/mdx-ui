## 7. Register Components in Astro

Create:

```text
src/components/mdx-ui/mdx-components.ts
```

Example:

```ts
export const mdxComponents = {};
```

Update your Astro configuration:

```js
import { defineConfig } from "astro/config";
import react from "@astrojs/react";
import mdx from "@astrojs/mdx";

export default defineConfig({
  integrations: [react(), mdx()],
  markdown: {
    remarkPlugins: [],
  },
});
```

Components are passed per-page in Astro via:

```astro
---
import { mdxComponents } from "../components/mdx-ui/mdx-components";
const { Content } = Astro.props;
---

<Content components={mdxComponents} />
```
