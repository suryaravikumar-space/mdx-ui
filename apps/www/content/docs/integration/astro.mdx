## 2. Add React Integration

MDX UI components are built with React, so you need to add the Astro React integration.

```bash
pnpm astro add react
```

## 3. Add MDX Integration

Install the Astro MDX integration.

```bash
pnpm astro add mdx
```

## 4. Initialize MDX UI

Initialize MDX UI in your Astro project.

```bash
pnpm dlx @ravikumarsurya/mdx-ui init
```

## 5. Add Components

Install the required MDX UI components.

```bash
pnpm dlx @ravikumarsurya/mdx-ui add callout accordion
```

## 6. Register Components in Astro

Create:

```text
src/components/mdx-ui/mdx-components.ts
```

Example:

```ts
export const mdxComponents = {};
```

Update your Astro configuration:

```js
import { defineConfig } from "astro/config";
import react from "@astrojs/react";
import mdx from "@astrojs/mdx";
import { mdxComponents } from "./src/components/mdx-ui/mdx-components";

export default defineConfig({
  integrations: [react(), mdx()],
  markdown: {
    remarkPlugins: [],
  },
});
```

Components are passed per-page in Astro via:

```mdx
<Content components={mdxComponents} />
```
